# Control Notas - Examen Final

Proyecto desarrollado para la asignatura **Programación Front End** utilizando Vue.js, HTML, CSS y JavaScript

## Descripción

La aplicación permite:

- Calcular calificaciones ponderadas
- Validar notas y asistencia
- Determinar si un estudiante aprueba o reprueba
- Registrar usuarios mediante un formulario validado
- Navegar entre distintas secciones mediante pestañas
- Visualizar una interfaz responsive adaptable a diferentes dispositivos

## Funcionalidades

### Cálculo de calificaciones

- Nota 1 (35%)
- Nota 2 (35%)
- Nota 3 (30%)
- Asistencia

#### Reglas de aprobación

- Promedio ponderado mayor o igual a 40
- Asistencia mínima de 80%

#### Validaciones

- Notas entre 10 y 70
- Asistencia entre 0 y 100

### Formulario de Registro

- Nombre
- Correo electrónico
- Contraseña
- Repetir contraseña

#### Validaciones

- Campos obligatorios
- Formato correcto de correo electrónico
- Contraseñas coincidentes

## Tecnologías utilizadas

- Vue.js 3
- JavaScript
- HTML5
- CSS3
- Vite
